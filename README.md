# Strapi Blog

A blog made with Reactjs and Strapi to learn how to work with that CMS.



## Tech Stack

ReactJS, Vite, Tailwind, Strapi and Supabase.




## Installation


```bash
  git clone https://github.com/riordansantos/strapi-blog.git
  cd strapi-blog
  code .
  npm run dev
```
